# plato-dashboard

Fleet dashboard rendering for PLATO nervous system.

## What It Does

Renders a real-time fleet dashboard showing room states, autonomy metrics, signal chain throughput, and cross-room coordination status across all PLATO rooms.

## Ecosystem

- **[plato-state](https://github.com/SuperInstance/plato-state)** ← Depends on (room state vectors)
- **[plato-autonomy](https://github.com/SuperInstance/plato-autonomy)** ← Depends on (autonomy metrics)
- **[plato-coordination](https://github.com/SuperInstance/plato-coordination)** — Fleet coordination status
- **[plato-nervous](https://github.com/SuperInstance/plato-nervous)** — Signal chain metrics

See [DEPENDENCIES.md](./DEPENDENCIES.md) for the full dependency map.
